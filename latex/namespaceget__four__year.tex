From a url that contains a four year plan process it and produce a JSON file in a structure that we can utilize to then convert into a clingo file. 

For the page in the computer science four year plan, create a JSON file with the class content organized by year and semester. This data is then used to discover conflicts that are higher priority.

@params url A string containing the url for the page with the four year plan @params output\+\_\+file A string that contains the file path and filename that the JSON will be written to.
