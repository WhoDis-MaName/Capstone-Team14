Run the clingo program for identifying conflict. 

The conflict identifier randomly chooses one section from each course to be called "{}critical"{}. We then define conflict such that two sections have overlapping times, overlapping days, and are not sections of the same class.

We then define a critical conflict to be if two critical sections of the same year (ie, two 3000 level courses) are in conflict. Since critical sections are chosen randomly, we then minimize the count of critical sections, finding the best selection.
