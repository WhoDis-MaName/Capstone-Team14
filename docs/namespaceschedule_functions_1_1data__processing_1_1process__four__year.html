str semester )

Get the list of classes for a particular semester from the four year plan, assuming that there may be multiple plans given.

Parameters
plan

A list of dictionaries. Each dictionary defines a four year plan such that it has this structure

{ 'year':{ 'semsester':[ [ list of class numbers], [ list of class names], number of credits filled by the class(es) in the list ] ... } ...
}

Parameters
yearA string or an integer to identify the year in the plan that is to be read from. Options: ['First Year', 'Second Year', 'Third Year', 'Fourth Year'] or [1, 2, 3, 4]
Parameters
semesterA string that determines if fall semester or spring semester Options: ['FALL', 'SPRING'] or any variation in capitalization
Returns
A list of classes in the same structure as origional semester list combined for all plans in the origional list.
