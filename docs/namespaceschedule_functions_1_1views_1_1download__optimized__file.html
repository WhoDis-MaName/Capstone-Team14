scheduleFunctions.views.download_optimized_file.download_optimized_file ( request,
filename )

Handles the download of an optimized file.

This view attempts to locate and return the requested file for download. If the file exists in the specified directory, it is served as an attachment. If the file does not exist, a 404 error response is returned.

Parameters
requestThe HTTP request object, which includes the 'filename' parameter as part of the URL.
filenameThe name of the file to be downloaded. This should correspond to a file located in the 'uploads' directory under MEDIA_ROOT.
Returns
A FileResponse to serve the file as an attachment if it exists, or a JsonResponse with an error message and a 404 status if the file is not found.
